| Workspace                       | Proves                                          |

| `examples/arktype-node-rollout` | ArkType, positionals, plugins, lifecycle hooks  |

Run an example from its workspace:

```bash
cd examples/zod-node-release
RELEASE_DEPLOY_REPLICAS=4 node src/index.ts deploy production --image=ghcr.io/acme/web
```

Completion commands currently return placeholder metadata:

```bash
node src/index.ts completion zsh
```
